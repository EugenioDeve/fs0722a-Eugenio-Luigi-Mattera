Come creare un template per i post

di Eugenio mattera - pubblicato il 25-10-2020

La prima cosa da fare è suddividere in blocchi logigi la pagina, identificando le parti comuni da applicare a tutte le pagine

    Le varie aree saranno quindi suddivise in:

  • Contenuti di intestazione del tag header
  • Contenuti centrali variabili del tag main(corpo principale)
  • Contenuti di fine pagine comuni, da inserire nel tag footer

Formattazione del codice Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezioneare con CSS. Perciò i passi saranno:

  1. Progettare il markup assegnando ad ogni specifico testo il tag appropriato
  2. Scrivere l'HTML dai tag esterni a quelli interni e dall'alto verso il basso
  3. Applicare stili CSS

L'applicazione di stili CSS è cresciuta negli anni fino a raggiungere la diffusione pressocchè totale
